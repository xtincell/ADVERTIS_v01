## 🚀 How to Use

### View Locally

1. **Unzip the archive** to any folder on your computer
2. **Open `index.html`** in any modern web browser (Chrome, Firefox, Safari, Edge)
3. **Navigate** between documents using the document switcher or links
4. **Search** within each document using the search bar
5. **Export** individual documents to PDF using your browser's Print → Save as PDF

### Deploy to Web

Upload the entire folder to any web hosting service:
- **Netlify / Vercel:** Drag & drop the folder (free hosting)
- **GitHub Pages:** Commit to a repo, enable Pages
- **Traditional hosting:** Upload via FTP to your web server

No build process required — pure HTML/CSS/JS!
